# Trigram-Language-Model

## How to Run
1. Place your corpus file in the project folder (example_corpus.txt is provided)
2. Run:

    python main.py

3. The generated text will be saved in:

    sample_output.txt

## Files
• model.py — trigram model implementation  
• utils.py — data loading and cleaning  
• main.py — trains the model and generates 100 words  
• evaluation.md — design decisions summary  
• example_corpus.txt — training dataset
